## ğŸš€ Getting Started

### Prerequisites

- [Shopify Partner Account](https://www.shopify.com/partners)
- [Theme Kit](https://shopify.github.io/themekit/) installed
- Basic knowledge of Liquid, HTML, CSS, and JavaScript

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shoaib-1965/Junaid_tele_com_shopify_web.git
   cd Junaid_tele_com_shopify_web
   ```

2. **Configure Theme Kit**
   ```bash
   theme configure --password=[your-api-password] --store=[your-store.myshopify.com] --themeid=[theme-id]
   ```

3. **Deploy to Shopify**
   ```bash
   theme deploy
   ```

4. **Watch for changes (Development)**
   ```bash
   theme watch
   ```
